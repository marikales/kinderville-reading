# Обучение чтению по методике Т. Мельниковой

**Курс для педагога.** Здесь собрано то, что нужно, чтобы понять метод и начать вести занятия — не заучить сценарии, а схватить логику и уметь построить урок самому.

> Методика Татьяны Мельниковой («Просто читай! Школа»). Русское чтение в Kinderville ведётся по ней. Материал собран из первоисточников: общие принципы автора, годовой курс из 38 уроков, тренажёры, консультация с автором.

## Суть в трёх фразах

- Ребёнок **читает с первого урока**: букв отдельно не учат — буква сразу живёт в сюжете и складывается в слог.
- Ядро — **слог** (слитно, одним движением рта, со смыслом), а не буква и не целое слово. Звук рождается из образа и действия, буква — из тела и геометрии.
- Всё новое — **по одному элементу за раз**; темп по возрасту.

## Как устроен курс: три части, три роли

Материалы Мельниковой — это **три разных инструмента**. Не путать их роли — тогда метод собирается:

| Часть | Что это | Зачем педагогу |
|---|---|---|
| **Общие принципы + этапы** | 9 этапов, правило «один элемент новизны», критерии перехода | **Скелет.** По нему строится любой урок. |
| **38 уроков (2–3 года)** | живой год игровых уроков | **Образец подачи.** Отсюда — *как* вести (звук из сюжета, три пространства). |
| **Тренажёры** | листы на отработку слоговых структур | **Автоматизация.** Для того, кто уже читает: скорость и точность. |
