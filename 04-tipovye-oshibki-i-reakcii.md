# Типовые ошибки и реакции педагога

> Для педагога. Две части: ошибки ДЕТЕЙ (что делать у доски) и ошибки ПЕДАГОГА (чтобы не сломать метод). Источники — инструкции тренажёров, «Общие принципы», приёмы из 38 уроков.
> ⚠️ Честно: развёрнутого «банка детских ошибок с реакциями» в материалах Мельниковой НЕТ — сценарии уроков почти линейны. Ниже собрано из редких прямых оговорок автора + выведено из логики метода (помечено). Это тот раздел, где сильнее всего нужна живая Таня / наблюдение за детьми — см. пометки «дособрать».

## Часть 1. Ошибки ДЕТЕЙ и что делать

### 1. Читает слог по буквам, «застревая» (Р… Ы… Б…, не РЫ)
- **Это главный барьер.** Цель метода — слог *одним движением рта*, а не сумма звуков.
- **Реакция:** вернуть телесный образ слога. Палец-персонаж «шагает по слогам» — на каждый шаг один слог целиком. Приём автора для гласной: «видишь У — губы в хобот, читай как слоник». Сначала слоги с *одинаковой* гласной, потом разводим.
- **Не переходить на след. этап**, пока слитности нет (критерий перехода — [02](02-posledovatelnost-kursa.md)).

### 2. Путает гласную в похожих слогах (ЖУК–МУК–ЛУК, читает наугад)
- Прямая оговорка автора: *«дети часто ошибаются при чтении гласной в похожих слогах»*.
- **Реакция:** привязать гласную к артикуляции-образу (У=хобот/гудит, А=широкий рот, И=улыбка). Тренажёр «3 буквы 5 историй» именно на это — многократное перечитывание близких слогов.

### 3. Угадывает слово по картинке/контексту, а не читает
- **Реакция:** заставить палец пройти *каждый слог по пути* (приём «слон шагает по слогам»). Тесты А5 специально дают похожие искажения (рыба/руба/быба) — если ребёнок «читает» картинку, он ошибётся, и это видно.

### 4. Называет букву, а не звук («эм», «ка» вместо [м], [к])
- **Реакция:** на всех ранних этапах — **только звук** ([ш], не «ша»). Буква — это «как звук выглядит». Имя буквы («эм») не нужно для чтения слога и мешает слиянию.

### 5. Теряет строку / читает справа налево или вразнобой
- **Реакция:** пальчик по «дорожке-строчке» строго слева направо (приём вводится в У15). Дорожка из «кочек»/«следов» — физический трек.

### 6. Ошибка на смысле («озвучил, но не понял»)
- **Реакция:** после прочтения — вопрос по прочитанному («что съел тукан?»). Если ответить не может — читаем ещё раз, восстанавливаем сюжет. Смысл — часть критерия «прочитал».

### 7. Не тянет мелкую/сюжетную нагрузку по возрасту
- **Реакция (прямо из уроков):** возрастная дифференциация. «До 3 лет — просто раскрашивают» (вместо рисования по точкам); «старшим — считать/дорисовывать лапы-палочки», «полукруг = часть круга» (младшим только образ). У23, У31 дают готовые варианты «для 3 лет / для 2 лет» — **бери как модель уровневой раздатки**.

> **Дособрать с Таней/из практики:** ошибки на йотированных (Е, Ё, Ю, Я), на мягком знаке, на стечении согласных (слон/слан), на оглушении (дуб→дуп). В материалах отдельного разбора нет.

## Часть 2. Ошибки ПЕДАГОГА (это ломает метод — читать до первого урока)

### О1. Вводит букву и новую структуру слога сразу
- Цитата автора: *«если появилась новая буква — одновременно поменять структуру слога будет ошибкой»*. **Один новый элемент за раз.** Новая буква → в знакомой структуре. Новая структура → на знакомых буквах.

### О2. «Проходит буквы» отдельными уроками на запоминание
- Метод прямо против этого: буква сразу в сюжете и в слоге, читаем с первого урока. Урок «учим букву Н» вне сюжета — не по методу.

### О3. Даёт имя буквы вместо звука
- «Эм», «ка», «эс» вместо [м], [к], [с]. На этапе слияния это враг №1.

### О4. Показывает букву раньше, чем прожит звук
- Сначала образ и звук (лес шелестит [ш]), ПОТОМ буква. Не наоборот. Если начал с «это буква Ш» — метод сломан.

### О5. Берёт для закрытого слога парный звонкий согласный
- В закрытом слоге парные звонкие оглушаются (АВ→[АФ], АД→[АТ]). Нельзя. **Исключение — Ж.** Отсюда В, Б, Д, Г, З не идут в ранние закрытые слоги — только Х, Ш, П, М, К, Ч, Ж, Р.

### О6. Гонит темп, игнорируя критерий слитности
- Пока слог не читается слитно и без опоры на смысл — на след. этап рано. «Знает буквы» ≠ «читает».

### О7. Убирает игру/сюжет ради «эффективности»
- Сюжет — не развлечение, а носитель звука и смысла. Без персонажа слог становится абстракцией, которую ребёнок 4–5 лет не удержит. Финал метода по автору — *«взаимный интерес и любовь к чтению»*, не скорость ради скорости.

### О8. Ведёт гласные не красным / не напоминает про «командную» роль гласной
- Гласные обозначаем **красным**; напоминаем: гласная «командует», как открыть рот. Мелочь, но она держит фонематическую опору.

### О9. Даёт ребёнку читать целую строку тренажёра без образца/дозы
- В тренажёре: мелкий шрифт читает взрослый; «иногда достаточно выделить первый слог»; одну строку перечитываем многократно, в т.ч. на время. Не «прочитай весь лист», а «эту строчку — и на секунды быстрее прошлого раза».

## Мини-памятка на стену (для педагога)
1. Звук → потом буква. Всегда.
2. Один новый элемент за раз.
3. Только звук [м], не имя «эм».
4. Слог — слитно, палец шагает по слогам.
5. Гласные — красным.
6. Закрытый слог — без звонких парных (кроме Ж).
7. Не спеши: слитно + по строке + со смыслом = можно дальше.
8. Игра держит смысл — не убирай её.
