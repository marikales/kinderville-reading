# Последовательность курса и карта букв

> Для педагога Kinderville. Тео, 2026-07-06. Скелет — из «Обучение чтению. Общие принципы»; наполнение — из 38 уроков и тренажёров.

## Девять этапов (канон автора — держи как позвоночник)

| Этап | Что читаем | Как выбираем материал |
|---|---|---|
| **1. Знакомство со звуками и буквами** | — | Идёт сквозь ВСЕ этапы, буквы добавляются постепенно. Отдельных уроков «на буквы» нет. |
| **2. Закрытые слоги (Г+С)** — АМ, ОП, УХ | гласный+согласный | (1) звук и буква однозначны; **парные звонкие нельзя** (АВ→[АФ]), кроме **Ж**; (2) слог несёт смысл: шуршащий, шипящий, чихающий, пыхтящий, рычащий. |
| **3. Открытые слоги (С+Г)** — МА, ПА | группируем **по гласной**, старт с **А, У** | Яркая артикуляция, герой-«озвучка» (робот, слон). |
| **4. Слова из 2 слогов с одной гласной** | муму, куку; каша, Маша, Саша | Сначала одинаковые слоги или фиксируем последний. Здесь же — первые предложения (я, мой). |
| **5–6. Двусложные слова с разными гласными** | туча, лужа, муха | Начать с ударного первого слога. |
| **6–7. Слова из 3 букв (С+Г+С)** | кот, кит, лис, сыр, дом | Этапы 5-6-7 могут меняться местами / идти параллельно. |
| **8–9. Слоги новых структур; слова из разных структур** | стечение согласных (слон, волк), Ь | Мостик: «от слов из 3 букв к словам с двумя согласными подряд». |

**Скорость — по возрасту.** Прямая цитата: *«дети 6 лет могут перейти от 1-го к 3-му этапу за один урок»*. Для нашего Reception (4–5) — уплотняем, но не пропускаем звуковой анализ.

## Критерий перехода на следующий этап (чтобы не гнать)
Ребёнок готов, когда одновременно:
- уверенно узнаёт несколько букв;
- **произносит слог слитно, не застревая** на каждом звуке;
- **двигается по строке слева направо**;
- не просто озвучивает, а **думает о смысле** — готов ответить на вопрос по прочитанному.

Пока слог читается «по звукам с застреванием» — на следующий этап рано, даже если буквы «знает».

## Как 38 уроков «Комплексной программы» ложатся на этапы
Программа для 2–3 лет = этап 1 (знакомство) + вход в этап 2 (закрытые слоги). Она НЕ доходит до открытых слогов и слов — это дальше, в тренажёрах. Опорные точки (по номерам уроков):

- **У1–13** — буквознакомство через образ (Ш, О, К, И, А), слоговые схемы-«лодочки», счёт слогов. Чтения слога и строки ещё НЕТ.
- **У14 — ПЕРВЫЙ закрытый слог АМ** («буква А — рот открыт, М — рот закрылся: АМ, как пасть волка»).
- **У15 — ПЕРВОЕ чтение по строке** (палец слева направо, называем звуки).
- **У17–29** — новые согласные Ч, П, Ж, Р, Й и закрытые слоги с ними (АП, АШ, ОШ, АЙ, ОЙ, УЙ, УЖ, АР, ОР, УР); попутно знак «+».
- **У30–38** — блок гласных: А, О, У, Ы (30–34) → И, Э (35–36) → Я (38); на каждом уроке читаются закрытые слоги. К концу — 7 гласных, закрытый слог автоматизирован, чтение по строке освоено.
- ⚠️ **Выпускного урока в программе нет** — уроки 35–38 по формату идентичны 30–34. Программа обрывается на «закрытые слоги + гласные», НЕ доводя до открытых слогов. Дальше нужен переход на тренажёрную линию (открытые слоги → слова).

**Практический вывод для Reception:** 38 уроков дают богатейший банк *сюжетов и приёмов подачи* и доводят до уверенного закрытого слога. Открытые слоги и слова берём из тренажёров, ведя их тем же методом (сюжет + три пространства).

## Порядок введения букв (реконструкция)

**Согласные (по мере занятий, «одна новая буква на занятие»):**
- Ранние (озвучиваемые): **Ш, М, К, П, Ч, Ж, Р, Й, Х, Ф** — все дают смысловой слог (шуршит/пыхтит/чихает/жужжит/рычит).
- Через слова (нейтральные, позже): **С, Л, Г, З, Н, Б, Д, Т**.

**Гласные (канон «Этапов»):** первый ряд **А, О, У, Э, Ы** → закрепление 5 → второй набор **И, Я, Ю, Е, Ё** → закрепление 10. (Гласные всегда обозначаем **красным** — «гласная командует, как открыть рот».)

## Карта пяти «трудных» букв Н/В/С/Х/Е — где ВПЕРВЫЕ несут нагрузку
Это ответ на давний открытый вход атома. ⚠️ **Статус — реконструкция, не факт:** посадка выведена из полного курса-оглавления («Обучение чтению.docx») + тренажёрных листов + принципа «в закрытых слогах нет парных звонких». В самих 38 уроках слов Вова/Наташа/Саша/«ух» НЕТ — они в тренажёрной линии и docx-ТП. Уверенность высокая (принцип жёсткий), но **сверить у Тани** порядок В/Н. Посадка не произвольна:

| Буква | Где впервые работает (не карточка, а слог/слово) | Почему так |
|---|---|---|
| **Х** | **раньше всех** — закрытый слог **«ух»** (У+Х), «Что слышно в лесу» (сова кричит «ух»). Далее словами: муха, хурма, арахис, наверху. | Х глухой, «озвучиваемый» → годится для закрытого слога сразу. |
| **В** | **открытые слоги**, имя **«Вова»** (ВО-ВА). Закрепление — «Васька» со звуковой схемой [в]. | В — парный звонкий → **в закрытом слоге невозможен** (АВ→[АФ]). Только открытый/слово. |
| **С** | **открытые слоги**, **«Саша»/«СУП»**; массово в 3-буквенных: лес, лис, лось, рысь, сыр, нос. | Нейтральный, не «озвучивается» → вводится словами, не ранним слогом. |
| **Н** | **открытые слоги**, **«наша»/«Наташа»**; массово: луна, тукан, банан, дон/ном. | Нейтральный сонорный → тоже через слова. |
| **Е** | карточка рано (блок гласных); первое **чтение** — «ем» («Я ем мух»); в 3-буквенных «лес/нет/сер»; **целевой методический ввод — пара Е–Ё** в «не просто слова» (лист «е_ё», ёлка/елка). | Йотированная — сложность, поэтому осмысленный ввод отложен к паре Е–Ё. |

**Свод для фоника-таймлайна:** к концу закрытых слогов у ребёнка есть **Х** (и русская опора для латинской X). **В, С, Н** приходят на этапе открытых слогов/слов — это середина Y1 по нашей раскладке. **Е** осмысленно — поздно (пара Е–Ё). Это уточняет §4.2-Т: из пяти «двойников без известной посадки» (B/C/H/E/X) русская опора по **Х** есть рано, по **В, С, Н** — на открытых слогах, по **Е** — поздно. См. [05-vhody-i-styk-s-anglijskim.md](05-vhody-i-styk-s-anglijskim.md).

## Диагностика
Готовые инструменты в комплекте: **«Чтение. Тесты. А5»** (листы: выбрать настоящее слово среди похожих искажений — проверяют *точность декодирования*, что ребёнок читает, а не угадывает; сложность растёт по слоговой структуре) и демо-тест «Просто слоги». Числовых порогов автор в тестах не задаёт — критерий перехода берём из списка выше («слитно, по строке, со смыслом»). Скоростной замер — приём из тренажёра (читать строку на время, фиксировать секунды, радоваться каждой).
